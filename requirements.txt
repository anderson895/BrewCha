# BrewCha — Python dependencies
#
# This file is intentionally empty. The project has NO pip dependencies.
#
# Why?
#   - serve.py uses only the Python standard library
#     (http.server, socketserver, webbrowser, os, sys)
#   - main.py runs INSIDE THE BROWSER via PyScript/Pyodide.
#     Its imports (pyscript, pyodide.ffi, js, html) are provided by
#     the PyScript runtime loaded from the CDN in index.html, NOT
#     by pip. Installing them locally has no effect.
#
# If you later add a backend (Flask, FastAPI, etc.) or dev tools
# (black, ruff, pytest), list them below — one per line — then run:
#     .\venv\Scripts\pip install -r requirements.txt
#
# Example:
#   flask==3.0.0
#   ruff==0.4.0
